About the Website

This website is based on student management system in which admin can insert, delete, and search a student. It also has a contact us section in which details filled in the form will send to the Web developer who has developed this website.

Features of the website:-

  1. Simple User Interface.
  2. Admin can start only by logging in.
  3. Insertion is very easy just by filling a simple form.
  4. Deletion is also easy by just a single click.

Software used in this website:

  1. Visual Studio (Text Editor).
  2. Chrome Browser for output.
  3. W3 css (CSS file of w3 school).
  4. Fontawesome Icons (for icons).
